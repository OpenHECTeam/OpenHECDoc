OpenHEC FPGA云

OpenHEC（Open ecosystem for High-Efficiency Computing）是虎甲虫公司以“互联网+ FPGA”为手段倾力打造的线上线下 一体化的产学融合FPGA创客生态圈。用户可线上完成产品设计、开发、验证，验证好的云节点可直接用于线下产品的快速实现。支 持高校实验教学、竞赛、科研与企业产品研发、创业更便捷地融合，并以众包方式让FPGA更简单、更便利、更便宜。
